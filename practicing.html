Input語法說明
input值 描述
button 定義可點及按鈕(多數情況下，用於通過JavaScript啟動腳本)
checkbox 定義複選框
file 定義輸入字段和"瀏覽"按鈕，供文件上傳。
hidden 定義隱藏的輸入字段。
image 定義圖像形式的提交按鈕。
password 定義密碼字段，該字段中的字符被掩碼。
radio 定義單選按鈕。
reset 定義重置按鈕。重置按鈕會清除表單中的所有數據。
submit 定義提交按鈕。提交按鈕會把表單數據發送到服務器。
text 定義單行的輸入字段，用戶可在其中輸入文本。默認寬度為20 個字符。
